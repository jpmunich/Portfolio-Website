About (Uninked Projects and Linked Portfolio Below)

Hello! I am Parker Coyle and this website is meant to showcase my improvement and overall journey as a web developer/programmer. I began my journey in the winter of 2021 when I discovered how to build static web pages in HTML. From there, I learned to make my static sites look better using CSS. This led me into making interactive and dynamic pages with Javascript. I began with small projects using the language before learning how to use data structures such as objects and modules to improve organization. I have since further branched out and learned to use technologies such as webpack to compile packages (external tools) easily and github to maintain version control within my projects. To spice up my frontend skills, I learned react to speed up development and handle tasks with a higher overhead of complexity to them. I have also ventured into pure computer science and data structures and algorithms programming through solving leetcode problems and by learning new languages such as C and C#. This made me much more efficient in my work and I can now sustain much more functionality with less code. The next step I want to take in my endeavors is to learn NodeJS and build full stack projects then to delve into machine learning with Python. I plan to build bigger and faster projects with an intention to use programming to help solve real-world problems.
